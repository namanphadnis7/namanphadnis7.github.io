AimBot

Aimbot is a slingshot-style target aiming game where the player must shoot their soccer ball at the target to clear the level.

The game was developed in GameMaker Studio using GML scripts.

I made this game as part of a junior year class. I drew all the game assets myself in the GameMaker sprite maker.

The first iteration of the game worked only by clicking at an angle with a fixed power. Later, the mousehold power option was added.

The player must overcome several obstacles alongside getting the timing and the aim of the shot correct in order to win!
